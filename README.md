#  فكرة المشروع

موقع تعليمي يساعد المبتدئين على تعلم تصميم مواقع الويب باستخدام:

## HTML
تستخدم لبناء هيكل صفحات الموقع وإضافة النصوص والصور والأزرار.

## CSS
تستخدم لتنسيق الموقع وإضافة الألوان والخطوط وترتيب العناصر.

## JavaScript
تستخدم لإضافة التفاعل والحركة داخل الموقع مثل الأزرار والتنبيهات والتأثيرات.


ويحتوي الموقع على:
- صفحة رئيسية
- قسم الدورات
- قسم الأسعار
-JavaScript تأثيرات وتفاعلات باستخدام
